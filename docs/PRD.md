# 문정동 베이커리 추천 시스템 PRD

## 개요

문정동 지역 베이커리를 분위기·목적·가격대 조건으로 필터링하고, 리뷰 기반 특징 분석을 통해 TOP 3 베이커리를 추천하는 웹 서비스.

---

## User Story

### 베이커리 추천

- 사용자는 분위기(아늑한/모던한/빈티지한/감성적인/편안한), 목적(브런치/선물/모임/케이크/빵구경), 가격대(저가/중가/고가) 중 원하는 조건을 선택할 수 있다.
- 사용자는 조건을 선택하지 않아도 평점 기준 TOP 3 베이커리를 추천받을 수 있다.
- 사용자는 추천 결과에서 베이커리 이름, 주소, 대표 메뉴, 평점, 가격대, 특징 태그를 확인할 수 있다.
- 사용자는 결과 페이지에서 "다시 추천받기"를 눌러 조건을 변경할 수 있다.

### 베이커리 목록 조회

- 사용자는 전체 베이커리 목록을 조회할 수 있다.
- 사용자는 특정 베이커리의 상세 정보를 조회할 수 있다.

### 리뷰 기반 특징 분석

- 시스템은 베이커리 리뷰 텍스트에서 키워드를 추출하여 특징 태그(예: 빵맛집, 케이크맛집, 선물하기좋은)를 자동 생성한다.
- 사용자는 추천 결과에서 리뷰 기반 특징 태그를 확인하여 베이커리 선택에 참고할 수 있다.

## Functional Requirements

### FR-1. 베이커리 추천

| ID | 요구사항 |
|----|---------|
| FR-1.1 | 분위기, 목적, 가격대 조건을 조합하여 추천 요청할 수 있어야 한다 |
| FR-1.2 | 각 조건은 선택 사항이며, 미입력 시 해당 조건을 무시한다 |
| FR-1.3 | 점수 계산: mood 일치 +2, purpose 일치 +2, price_range 일치 +1, rating × 0.5 가산 |
| FR-1.4 | 점수 내림차순 정렬 후 상위 3개를 반환한다 |
| FR-1.5 | 조건에 매칭되는 베이커리가 없어도 rating 순으로 TOP 3을 반환한다 (빈 결과 없음) |

### FR-2. 베이커리 데이터 조회

| ID | 요구사항 |
|----|---------|
| FR-2.1 | GET `/api/bakeries` — 전체 베이커리 목록을 JSON으로 반환한다 |
| FR-2.2 | GET `/api/bakeries/{id}` — 단일 베이커리를 JSON으로 반환한다 |
| FR-2.3 | 존재하지 않는 id 조회 시 404 응답을 반환한다 |

### FR-3. 리뷰 기반 특징 분석

| ID | 요구사항 |
|----|---------|
| FR-3.1 | 리뷰 텍스트에서 사전 정의된 키워드를 매칭하여 태그를 생성한다 |
| FR-3.2 | 키워드-태그 매핑: 아늑한, 빵맛집, 케이크맛집, 마카롱맛집, 가성비좋은, 줄서는집, 선물하기좋은, 브런치맛집 |
| FR-3.3 | 리뷰가 없는 베이커리는 빈 태그 리스트를 갖는다 |
| FR-3.4 | 태그는 앱 시작 시 자동 생성된다 |

### FR-4. 웹 프론트엔드

| ID | 요구사항 |
|----|---------|
| FR-4.1 | GET `/` — 조건 선택 폼 페이지를 렌더링한다 |
| FR-4.2 | POST `/recommend` — 폼 제출 시 TOP 3 추천 결과 페이지를 렌더링한다 |
| FR-4.3 | 결과 페이지에 베이커리 이름, 주소, 설명, 대표 메뉴, 가격대, 평점, 특징 태그를 표시한다 |
| FR-4.4 | 결과 페이지에서 메인 페이지로 돌아갈 수 있는 링크를 제공한다 |

## Non-Functional Requirements

| ID | 구분 | 요구사항 |
|----|------|---------|
| NFR-1 | 테스트 | pytest 기반 TDD로 개발한다. 모델, 추천 로직, 리뷰 분석, API 각각에 대한 테스트를 작성한다 |
| NFR-2 | 테스트 | 모든 테스트는 외부 의존성 없이 독립 실행 가능해야 한다 |
| NFR-3 | 성능 | 인메모리 데이터 기반으로 추천 응답 시간 100ms 이내 |
| NFR-4 | 구조 | FastAPI 라우터를 도메인별로 분리한다 (bakeries, recommend) |
| NFR-5 | 데이터 | MVP 단계에서는 DB 없이 인메모리 시드 데이터를 사용한다 |
| NFR-6 | 호환성 | Python 3.11 이상 |
| NFR-7 | 의존성 | 최소 패키지만 사용: fastapi, uvicorn, jinja2, python-multipart |
